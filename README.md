# AAU Department Admission Predictor

A beginner-friendly Flask web app to predict the chance of freshman students getting their preferred department at Addis Ababa University. It collects senior and freshman data, calculates competition and cut points, and provides a simple probability estimate using beginner-level ML concepts.

**Features:** Senior Survey (GPA, admitted/preferred department, cut points, reasons for not getting preferred department), Freshman Survey (GPA, department choices, number of seats), Prediction (estimates admission probability based on senior data), Cut Point Estimation.

**How to Run:** Clone/download repo → install packages `pip install flask pandas scikit-learn` → run `python app.py` → open browser at `http://127.0.0.1:5000/`.
